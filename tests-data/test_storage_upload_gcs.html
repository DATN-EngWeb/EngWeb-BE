📤 Test Presigned URL Upload (GCS)

FE gửi file trực tiếp lên Google Cloud Storage với signed URL

Quy trình Signed URL (FE gửi trực tiếp lên GCS):
1️⃣ FE xin signed URL từ BE (request-presigned-url)
2️⃣ BE trả về signed URL + required headers
3️⃣ FE PUT file trực tiếp đến GCS với signed URL
4️⃣ GCS kiểm tra signature và lưu file
5️⃣ FE xác nhận upload với BE (confirm-upload)
6️⃣ BE trả về file URL công khai

Cấu trúc folder:
• Tất cả files của test: media/tests/test_{test_id}/
Avatars: JPG, PNG (max 50MB)
Tests: JPG, PNG, MP4, MP3 (max 50MB)
HTTP Requests / Responses
